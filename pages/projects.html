> GPU path tracer on OpenCL

This project was my Bachelor`s work and I spare lot of f-ing time on it and I still don`t know how something works in it because of crazy maths and physics.. but it works.

Path tracing is the computer algorithm that allows us to get the physically based rendered image. It is embarrassingly parallel algorithm, so it is advisible to perform all calculations on a GPU device. OpenCL technology was chosen because it`s cross-platform.
