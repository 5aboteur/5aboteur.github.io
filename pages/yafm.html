YAFM

Overview

YAFM (Yet Another File Manager) – is a file manager, built for education purposes. It`s written in C language and uses ncurses library to provide the simple graphical user interface.

This is how the program might look after the first launch.


The manager uses built-in YATE editor. When you click on a text file, YATE launches automatically and shows the contents of the file.

YAFM has 3 abstract sections. The upper one has all of YAFM features, such as file manipulations, extra options etc. Each option is accompanied with a function key. So, when you press the appropriate key – YAFM executes an option. All of them also have combination analogues like, for example, if we want to call the 'Help' window, we can press 'F7' button or use 'CTRL+I' shortcut.
