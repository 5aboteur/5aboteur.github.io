YATE (Yet Another Text Editor) â€“ is a text editor, built for education purposes. It`s written in C language and uses ncurses library to provide the simple graphical user interface.

The basic launch of the program looks something like this.


Here we have 3 sections of the program. The upper section has all YATE options. You should press an appropriate function key to get one of the options. All of them also have "control key combo" analogues, so if we want to open the file, we can press 'F4' button or use 'CTRL+O' shortcut. YATE also has some extra features. For example, if we want to change the current theme, we can use 'CTRL+G' combination.


Middle section contains the text that you wrote or got from the file. The bottom section displays some useful (or not so?) information about the file and current state of the editor (e.g. is live encryption enabled, errors, version etc).
